# Preferências e Padrões — LINKA

- **Idioma**: Português de Portugal (PT-PT).
- **Estilo de Código**: Clean Code, nomes descritivos em português.
- **Segurança**: JWT para autenticação, Bcrypt para passwords.
- **Desenvolvimento**: Ordem rigorosa (BD -> Backend -> Segurança -> Sockets -> Testes -> Frontend).
