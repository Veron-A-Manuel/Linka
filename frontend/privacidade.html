Política de Privacidade

1. Dados que Recolhemos

Quando utiliza a Linka, podemos recolher os seguintes tipos de dados:

1.1 Dados de Registo

  • Nome completo
  • Endereço de email
  • Número de telefone
  • Password (armazenada encriptada com bcrypt)

1.2 Dados de Perfil (Opcionais)

  • Fotografia de perfil
  • Cidade e bairro
  • Coordenadas GPS (apenas se autorizado)

1.3 Dados de Actividade

  • Histórico de anúncios publicados
  • Histórico de compras e vendas
  • Mensagens no chat
  • Avaliações dadas e recebidas
  • Favoritos
  • Denúncias efectuadas

1.4 Dados Técnicos

  • Endereço IP
  • Tipo de dispositivo e navegador
  • Dados de geolocalização (quando autorizado)

2. Como Utilizamos os Seus Dados

Os seus dados são utilizados para:

  • Prestação do serviço: Processar transacções, gerir a sua conta e comunicar com outros utilizadores
  • Segurança: Prevenir fraudes, proteger contas e cumprir obrigações legais
  • Melhoria: Analisar o uso da plataforma para melhorar a experiência
  • Comunicação: Enviar notificações sobre a sua actividade na plataforma
  • Marketing: Enviar informações sobre novidades (apenas com consentimento)

3. Partilha de Dados

A Linka não vende os seus dados pessoais a terceiros. Podemos partilhar dados apenas nas seguintes circunstâncias:

  • Com outros utilizadores: Nome, telefone e localização quando realiza uma transacção (apenas as informações necessárias para a venda)
  • Com prestadores de serviços: Empresas que nos ajudam a operar (hosting, email, analytics), sob acordos de confidencialidade
  • Por obrigação legal: Quando exigido por lei ou por ordem judicial
  • Protecção de direitos: Para defender os direitos e segurança da Linka e dos seus utilizadores

4. Armazenamento e Segurança

  • Os dados são armazenados em servidores seguros com encriptação
  • Passwords são encriptadas com bcrypt (12 rounds)
  • Conexões são protegidas com SSL/TLS
  • Acesso aos dados é restringido apenas a pessoal autorizado
  • Dados de sessão utilizam tokens JWT com expiração curta (15 minutos)

5. Os Seus Direitos (LGPD)

De acordo com a Lei de Protecção de Dados Pessoais de Moçambique, tem o direito de:

  • Acesso: Solicitar uma cópia dos seus dados pessoais
  • Rectificação: Corrigir dados incorrectos ou incompletos
  • Eliminação: Solicitar a eliminação dos seus dados pessoais
  • Oposição: Oponhar-se ao processamento dos seus dados para certos fins
  • Portabilidade: Receber os seus dados num formato estruturado
  • Retirada de consentimento: Retirar o consentimento a qualquer momento

Para exercer estes direitos, contacte-nos em: privacidade@linka.co.mz

6. Retenção de Dados

  • Dados de conta: Mantidos enquanto a conta estiver activa
  • Dados de transacções: Mantidos por 5 anos (obrigação fiscal)
  • Dados de chat: Mantidos por 2 anos após a última mensagem
  • Dados de conta eliminada: Removidos em até 30 dias, exceto dados obrigatoriamente retidos por lei

7. Cookies e Tecnologias de Rastreamento

Utilizamos cookies para melhorar a sua experiência. Para mais detalhes, consulte a nossa Política de Cookies.

8. Menores de Idade

A plataforma é destinada a utilizadores com mais de 18 anos. Não recolhemos intencionalmente dados de menores de 18 anos. Se descobrirmos que recolhemos dados de um menor, eliminaremos esses dados imediatamente.

9. Alterações a Esta Política

Podemos actualizar esta Política de Privacidade periodicamente. As alterações serão publicadas nesta página com a data de actualização actualizada. Recomendamos que consulte esta página regularmente.
